# 1. 快读

```c++
inline int read(){
    int re=0,k=1;
    char ch=getchar();
    while(ch>'9'||ch<'0'){
        if(ch=='-'){
            k=-1;
        }
        ch=getchar();
    }
    return re*k;
}
```

> \_\_int128 占 16 字节，可存的数据范围达到了 long long 的平方: $ 10^32 $，不能用 cin，需要用手写快读快写（其实也可以用 scanf 和 printf）

```c++
// 快读
__int128 read(){
    __int128 x=0,f=1;
    char ch=getchar();
    while(!isdigit(ch)&&ch!='-')ch=getchar();
    if(ch=='-')f=-1;
    while(isdigit(ch))x=x*10+ch-'0',ch=getchar();
    return f*x;
}
// 快写
void print(__int128 x){
    if(x<0)putchar('-'),x=-x;

}
```
